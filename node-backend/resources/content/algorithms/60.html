What is a Stack? Stack is exactly what it sounds like, a stack of things. For example a stack of mail. You notice how  when you get your mails, you stack them on top of each other, and usually when you check them you go from the top. So the most oldest mail is found at the bottom, and you read the newest one.

This concept is known as way of getting data form the stack is also known: LIFO (Last in First Out) 
Key Methods:
  • Push - Add new data to the top of the stack
  • Pop - Get and remove the data from the top 
  • Peak - Fetch the data from the top but do not remove it



Implementing your own Stack:
This is just for better understanding how it works and a simplification of the Data Structure, it is not recommended that you use/implement your own Stack, as there is already an implementation of Stack (which is more safe from runtime issues) from whatever language you are using.
package algorithm.datastructures;

public class Stack {
    private String[] data;
    private int top;

    public Stack(int maxSize){
        data = new String[maxSize];
        top = -1; // at this point we do not have anything yet
    }

    public void push(String value){
        top++; //increment top to signify newest data
        data[top] = value;
    }

    public String pop(){
        String value = data[top];
        top--; // decrement top
        return value;
    }

    // do not increment o decrement, just check if the value is there
    public String peak(){
        return data[top];
    }

    public static void main(String[] args){
        Stack mailStack = new Stack(5);
        mailStack.push("Electricity Bills");
        mailStack.push("Rental Bills");
        mailStack.push("Water Bills");
        mailStack.push("Love Letter");
        System.out.println(mailStack.pop()); // Love Letter
        System.out.println(mailStack.pop()); // Water Bills
        mailStack.push("Government Letter");
        System.out.println(mailStack.peak()); // Government Bills
        System.out.println(mailStack.pop()); // Government Bills
        System.out.println(mailStack.pop()); // Rental Bills
    }


}

Example Implementation Use Case - reverseString:
Problem: Print out the letters in a word in reverse order, for example Hello should print out: o l l e H

Solution (using java 8)
java.util.Stack<Character> characters = new java.util.Stack<>();
String word = "Hello";
List<Character> list = word.chars() // convert string to a int stream (no direct stream call)
        .mapToObj(intValue-> (char) intValue) // convert each int value back to character
        .collect(Collectors.toList()); // collect them to a list agvain
characters.addAll(list);
while(!characters.isEmpty()){
    System.out.println(characters.pop());
}
