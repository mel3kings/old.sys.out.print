Procedure : LINEAR-SEARCH(a,x)
Inputs
  • a - the array to search in
  • x - the value we are for in a
Outputs
Index position where a[i] == x or-1 if not found

Very straight forward, just iterates over a list and return is its there. 
Effiiciency of Algorithm
Linear - O(n)

package algorithm;

public class LinearSearch {
    
    public static void main(String[] args){
        int[] a = {3,5,79,13,789,12,6,8};
        int x = 12;
        System.out.println(search(a,x));
    }

    public static int search(int[] a, int x){
        for(int i = 0; i< a.length; i++){
            if(a[i] == x){
                return i;
            }
        }
        return -1;
    }
}
