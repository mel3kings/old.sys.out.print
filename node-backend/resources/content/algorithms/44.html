Prime Number

Given a number n, check if this number is a prime:

E.g:
  • 15 returns false
  • 13 returns true

Solution
 
public static boolean isPrime(int n) {
 if ((n > 2 && n % 2 == 0) || n == 1) {
 return false;
 }
 for (int i = 3; i <= (int) Math.sqrt(n); i += 2) {
 if (n % i == 0) {
 return false;
 }
 }
 return true;
}

Get all the sum of all the digits in the given number.

e.g
  • 15 returns 6 (1+5)
  • 34 returns 7 (3+4)

Solution:
public static int summationOfValues(int input) {
 int sum = 0;
 int copyInput = input;
 while (copyInput > 0) {
 sum = sum + copyInput % 10;
 copyInput = copyInput / 10;
 }
 return sum;
}
