Problem with working with an array is that it has a fixed size and you can’t reorder the elements without traversing the whole array. Say you have the following array:

int[] numbers= new int[]{1,2,4,5,6,7,8,9,10};

And you realise, you forgot to insert number 3. You can’t just go:

numbers[2] = 3;

This will in turn just overwrite the number 4, but your will still have the wrong data than you are expecting.

What is Linked List
Linked List is Data Structure whereas every entries in the list contains a reference for the next one.
So for example below, A knows that B is next, and B knows C is next and so forth.

So, in our previously example should we want to insert 3, we would just need to update 2’s next node as 3, and 3’s next would be 4, and the “chain” would be correct.

Here is a crude representation of Singly Linked List:

package algorithm.datastructures;

public class SinglyLinkedNode {
    private String value;
    private SinglyLinkedNode nextSinglyLinkedNode;

    public SinglyLinkedNode(){}

    public SinglyLinkedNode(String value, SinglyLinkedNode nextSinglyLinkedNode){
        this.value = value;
        this.nextSinglyLinkedNode = nextSinglyLinkedNode;
    }
   
    public static void main(String[] args){
        SinglyLinkedNode finalNode = new SinglyLinkedNode("last node", null);
        SinglyLinkedNode node4 = new SinglyLinkedNode("4th node", finalNode);
        SinglyLinkedNode node2 = new SinglyLinkedNode("2nd node", node4);
        SinglyLinkedNode node1 = new SinglyLinkedNode("1st node", node2);

        SinglyLinkedNode node = node1;
        while(null != node){
            System.out.println(node.value);
            node = node.nextSinglyLinkedNode;
        }

        SinglyLinkedNode node3 = new SinglyLinkedNode();
        node3.value = "missed 3rd node";
        node3.nextSinglyLinkedNode = node4;

        node2.nextSinglyLinkedNode = node3;

        node = node1;
        System.out.println("Correct order");
        while(null != node){
            System.out.println(node.value);
            node = node.nextSinglyLinkedNode;
        }
    }
}

Notice how we have a reference always of the next node, and once we wanted to insert a new node, we didn’t have to check all the way to the last node. Also we did not need to worry about the size of the ‘chain’. 

This is obviously a crude representation, and only used for simplification.
