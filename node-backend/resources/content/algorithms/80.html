Address/Ports- As a server, can have several applications running on it, we need ports to specify which application will handle an incoming request.

DNS - Domain Name System
  • Allows a url to assign to an address with a default port
  • HTTPs usually defaults to port number 443 unless otherwise specified
For example:
When you go to google.com from you browser, a request is sent to their server (it could be a web server or another one)
Their server could be hosting different applications running, like a Node app, Email, FTP, etc.
Ports - enables to handle to delegate requests to which application running in a webserver
