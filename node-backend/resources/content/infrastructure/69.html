What is Terraform?
Basically, in a oversimplification of it, Terraform is Infrastructure as code. Similar to how Docker generates images and containers based on yml files, and how kubernetes manages it’s infrastructure  through config files, Terraforms enables you to create AWS instance (or other cloud provider) via config files.

Prerequisites
  • Knowledge and Access to AWS
  • AWS cli configured

Installing Terraform
Installing is Terraform for Mac OS is quite straightforward
brew install terraform 
brew is another Mac OS tool: https://brew.sh/ that is out of scope for this article
Windows requires download the package and adding it to Environment PATH : https://www.terraform.io/intro/getting-started/install.html

Verifying if it has been properly been installed, and initialising config:
terraform
terraform init

You will get something similar to:

Creating AWS Instances using Terraform:
Create yourself a new terraform file, it can be any name, for my case I’ve created a micro.tf file, that contains the following:

provider "aws" {
  access_key = ""
  secret_key = ""
  region     = "ap-southeast-2"
}

resource "aws_instance" "example" {
  ami           = "ami-43874721"
  instance_type = "t2.micro"
}

Provider block contains your IAM access keys, initially Terraform was intended for building AWS instances, but this can be change to other service provider as well. If you do not specify the credentials here Terraform will look in your .aws/credentails folder if one exists.  

The resource block is where the magic happens, basically you are telling terraform to create an “aws_instance” with “example” as the name with the following config
  • ami - is which image you want to base your instance form, make sure this is a valid AMI
  • Instance_type - is what kind of image you want it to be
  • The name “example” is just Terraform keeping track of the instance, this is NOT what it will be named in AWS

To actually perform the creation, run the following with the tf file in the same directory
terraform apply

Pitfall: Make sure there is only one tf file for now.
