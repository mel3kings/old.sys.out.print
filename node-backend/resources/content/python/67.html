Range
Built in function in python that is another way of iterating through a range of numbers for example:
for i in range(0,10): // 1st param is start, 2nd param is end, optional 3rd param is step
    print(i) // will print out 0-9

Enumerate:
Enumerate enables you to perform index counts in a form of tuples for any iterable object:
word = 'hello'
for item in enumerate(word):
    print(item)
will return:
(0, 'h')
(1, 'e')
(2, 'l')
(3, 'l')
(4, 'o')

Or you can also:
word = 'hello'
for index,letter in enumerate(word):
    print(f"index is {index}")
    print(f"letter is {letter}")

Zip
The opposite of Enumerate, pairs up items in forms of tuples:
mylist1 = [1,2,3]
mylist2 = ['a','b','c']
for item in zip(mylist1, mylist2):
    print(item)
will return:
(1, 'a')
(2, 'b')
(3, 'c')
Notes:
  • you can have more than two lists
  • you can assign them to a list, otherwise it will just return  a memory location
  • if the lists are uneven, it will only match until the shortest list
mylist1 = [1,2,3,4,5,6,7]
mylist2 = ['a','b','c']
numberlist = [100,200,300]
for item in zip(mylist1, mylist2, numberlist):
    print(item)

//will only return the below
(1, 'a', 100)
(2, 'b', 200)
(3, 'c', 300)

Min/Max
will return minimum or maximum from a list
min(mylist1) // will return 1
min(mylist2) // will return a
Pitfall: calling min/max to a combination of list with different data types will result into a runtime error:
TypeError: '<' not supported between instances of 'str' and 'int'

Random
Built in random library from python.
To import library using python you can do:
from random import shuffle // there are other different functions inside random other than shuffle
shuffle(mylist1) // will shuffle the values of a list

In place functions means it does not return any value, and the shuffling is done on the variable itself
https://docs.python.org/3.6/library/random.html

Input
asks for an input for the user
input('whats your name? ')
will make python to ask for an input from the user, you can assign the result then to a variable
Pitfall: Result will be in String, so if you are expecting an int, you have to cast it to appropriate data type

Flat out iterations
Usually when we do iterations we would have a for loop and then we want to store each value in a separate holder
Original way of doing it:
mystring='long world hello'
mylist =[]
for letter in mystring:
    mylist.append(letter)

Flat out iterations way:
mylist = [char for char in 'long word hello'] // one line way of doing it
Pitfall: syntax has to be <element> for <element> in <iterable>, element here is whatever variable you decide, but it needs to be the same

Advanced operations:
you can add an if statement at the end to perform some check, so the below will only list out even numbers from the range
mylist = [x for x in range(0,10) if x%2 == 0]
Another thing you can do is perform operations on it
mylist = [x**2 for x in range(0,10) if x%2 == 0]

The above code will square root each even number from range of 0-10
