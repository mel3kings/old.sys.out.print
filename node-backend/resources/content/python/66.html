Basic Comparator operations:
Python as per other programming languages has all the basic comparison operators:
  • > - less than
  • < - greater than
  • == - equal
  • != - not equal
  • and -
  • or
  • not
Sample usage:
not(1 == 1) and not(1 > 2) // True and False will return False
not 400 > 5000

Control Flow:
IF ELSE:
It is important to know that python uses indentation and colons to make use of control flow syntax. In contrast for example to Java, where we have brackets. Below are the basic Logic operators:
  • if
  • elif - else if
  • else

if some_condition:
    # execute this code // note the indentation
elif some_other_condition:
    # do something different
else:
    # do something else

For Loop:
Iterating or going through something, like a list of object. 
Syntax:
my_iterable = [1,2,3] // we have list of ints
for item_name in my_iterable: // item_name is the variable you want to declare
    print(item_name) 

More complicated iterations: 
mylist= [(1,2), (3,4), (5,6), (7,8)] // tuples inside a list
for (a,b) in mylist: // iterating and accessing the values inside
    print(f"first is {a}") // recall print f to access variables inside 
    print(f"second is {b}")

While Loop
Execute something while a condition is True
while boolean_statement:
    #execute code

Example:
hungry = True
food_counter = 0
while hungry:
    food_counter = food_counter + 1
    print("nom nom")
    if(food_counter == 10):
        hungry = False  

Loop Controls
  • break - break out of the loop
  • continue - continue to the next iteration
  • pass - pass and do nothing at all, used in avoiding runtime errors
