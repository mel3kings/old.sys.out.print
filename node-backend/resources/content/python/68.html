In Python, if you want to get documentation for built-in methods there are two ways to do it: 

1st way is to use the built-in help function:
help(mylist.pop) // notice no parameters as we are not invoking the method
Will return the below:
Help on built-in function pop:

pop(...) method of builtins.list instance
    L.pop([index]) -> item -- remove and return item at index (default last).
    Raises IndexError if list is empty or index is out of range.

Another way is visiting the documentation site:
https://docs.python.org/3/

Creating your own function:
Syntax:
Basic method syntax
def sayHello(name): //declare method
    '''this is a hello world function''' // documentation
    return ("Hello " + name) // return some values

Calling this method
greetings = sayHello('Mel') //assign the result to greetings variable
print(greetings)
help(sayHello) // this will return your documentation above

Setting a default value in parameters:
There are times where you want to set default values, for example the sayHello method above requires a name, but if this method is called without passing any parameter, it will throw an exception. So in this case we want to set a default value for the name:
 
def sayHello(name='Empty Void'):
    '''this is a hello world function'''
    return ("Hello " + name)

sayHello() // will return 'Hello Empty Void'

Variable Parameterization
*args 
There are instances where you do not know the number of values that your method parameters will take in
for example, you want to create a method that accepts integers and you want to add them all up, the problem is you don't know how many ints you want to add up

We can use the syntax below:
def addAllValues(*args): //args here are accepted as tuples
    print(sum(args))

addAllValues(1,1,1,1,23,21,3124,123,123,123,123) 

**kwargs
There are times you would want to use a dictionary as parameter, in this case we use the below syntax

def findMel(**kwargs):
    if 'Mel' in kwargs:
        print('I found Mel in {a}'.format(a =kwargs['Mel']))
    else:
        print('Mel not found')


findMel(April = 'Manila', Mel = 'Australia')
