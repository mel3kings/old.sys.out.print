Lists 
Basic list of objects, Python in particular is not very strict with the data types you use in a list. for example:

new_list = ['one', 1, 1.1 ]

Above is a list that contains three data types, String, int, float. This is not an issue with python. You can also append list to each other like so:

new_list = my_list + new_list

Indexing and slicing
Similar to how python handle Strings, we can do indexing and slicing
new_list[0] // returns the first element
new_list[1:] // returns first element until the end

Main difference is that list IS mutable, so you can change the values in specific indexes like so:
new_list[0] = 'new value'

Methods to Note:
  • append
  • pop - see Topic on Stack
  • sort - natural ordering 

Pitfall
Sorting does not return any value, sort does the sorting to the actual variable so the below will just return to you a NoneType, which is the Null equivalent to other languages
my_list = ['a','c','d','z' , 'f','k']
test = my_list.sort()
print(test)

What you can do is the below:
my_list.sort()
test = my_list
print(test)
