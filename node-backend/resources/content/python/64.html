There are several more data types in Python. Let's cover more of them now.

Dictionaries
dictionaries, are similar to Maps, whereas you have Key Values pairs. 
Instantiating a dictionary:
myDictionary = {'1':'1st place', '2': '2nd place'} //keys need to be Strings
myDictionary['1'] // access the the key 
myDictionary = {'1':'1st place', '2': [0,1,2,3], '3':1.1} //values can have different data types

Methods
  • Keys - returns the keys of a dictionary
  • Values - returns the values of a dictionary
  • Items - returns both key and value
myDictionary.items() // this command will retuun the below

Tuples
Are exactly the same as list, but instead they are immutable, which is the only difference. Tuples are created by using parenthesis as compared to list where you use braces.
t = (1,2,3)
type(t) // will return Tuple

Tuples help you ensure data integrity. As you gain more knowledge in programming you will know the value of having immutables.

Sets
Data type that ensures that the values are unique

initialized by:
t = set() // t will be the variable name
t.add(1) // adds some value
t.add('abc') // more value
t // print values
my_list = [2,2,2,2,2,2] // create a list with duplicate values
set(my_list) // cast to set, will filter to unique values only

Boolean
Use for control flow of your logic, the only thing to note in python is that it uses with upper case on the first letter so the only boolean allowed is True and False

1 > 2 // will return False

None
None data type is used to depict Null in Python so you can instantiate a variable and assign none for now and later assign a different value. otherwise it might throw an error. for example

b = None
b // will not print out anything but it will NOT throw an error as well.
