Basic String manipulation:
  • print - print line
  • len - length
  • indexing -
    myString = "Hello World"
    myString[1] //e
    myString[-1] //d - reverse indexing
        'Hello World'[8] //r - indexing directly from a string
  • slicing -
   myString[1:] // ello World
   myString[:3] // Hel //index exclusive
   myString[1:3] //el // 2nd param is exclusive
   myString[::2] // HloWrd // third param is getting the steps 

Pro Tip:
in interviews, if they ask you to reverse a String, and you can use any language, impress them with your python skills, using this one liner:
myString[::-1]

String properties and methods
 String are immutables, cannot be changed, you cannot the values of a string, but you can do String concatenation
myString = "Hello World"
myString[1] = "Z" // will throw the below error
    TypeError: 'str' object does not support item assignment
myString[0:] + " it is a good day" // valid statement   

Pitfall: When executing other functions, make sure you put a parenthesis, otherwise python will return to you a description of that method. for example:
myString.upper //will return: function str.upper 
myString.upper() // will return an uppercase version of your String

String Format 
Dynamically changing a String using format:
print('this is a {}'.format('TEST'))
print('The {2} {1} {0}'.format('fox', 'brown', 'quick'))
print('The {b} {q} {f}'.format(f='fox', b='brown', q='quick'))
   new format for python 3:
print(f'This is {myString}')
