Node Express - possibly one of the most popular librarys in node is express. 
  •  Express enables to create a web server really quite easily. (if you have worked with Java before, you'd know that there are alot of initial configuration needed to be done before you are able to set up a basic webserver)

Install Express:
npm install express --save        

Creating a hello world page in Node:
var express = require('express');
var app = express();

var port = process.env.PORT || 3000;

app.get("/", function(req, res){
    res.send('<html><head></head><body>Hello world</body></html>')
});
app.listen(port);

Breakdown:
var express = require('express');
var app = express();
 These first two lines, is required. we are importing the express library to our application, we then initialize the application by making a call to express();
We assign the application to the variable app, we then need to declare what the methods should be.

var port = process.env.PORT || 3000;
Our web application will listen to an environment variable called PORT, otherwise set it to default 3000.
In your local machine it will be localhost:3000

app.get("/", function(request, response){
   request.send('<html><head></head><body>Hello world</body></html>')
});

  • app.get - means if we receive an app HTTP GET Method Request, with the URL "/", so localhost:3000/
  • then respond with whatever we send to response
  • We take two parameters request and response, as per normally the case when we are handling http request
app.listen(port);
Finally we have the app listening on the port
