What is NodeJS?
If you have basic knowledge of a backend programming language plus basic javascript, you already have a sound foundation of knowing NodeJS without knowing it! NodeJS is a combination of javascript plus C++ as a backend engine, the v8 engine. It moves Javascript from client-side programming to server-side.

There are some key concepts from javascript that are core elements of Node that are not really obvious if you have not a deep knowledge of javascript. Below is a refresher for Javascript before diving deep to NodeJs

Javascript Concepts Refresher
First-Class Functions
Functions in Javascript are considered to be first-class, meaning they can be passed around as a parameter, and be invoked. 
function greet(){
    console.log('hi');
}
function logGreeting(fn){
    fn();
}
logGreeting(greet);
Here we have a function greet() and logGreeting() which take a parameter, and then we invoked the greet function inside another function

Anonymous Function
In this scenario we created a function with no name, but we assigned it to a variable, that in turn we then invoke.
var greetMe = function () {
    console.log('hello again!')
}

greetMe()

Objects in Javascript 
Not a well known fact but in Javascript objects are mostly key value pairs. JSON - in fact originates from Javascript Object Notation. JSON are Javascript's Objects. Similar to other Object oriented languages, the objects can have methods (or functions in this case).
 For Example, below are all JS codes:
var person = {
    firstname: 'Mel',
    lastname: 'Tatlonghari',
    greet: function(){
        console.log('Hello , ' + this.firstname + ' ' +
        this.lastname)
    }
}

person.greet()

You can also get the variable inside the object by brackets:
console.log(person['firstname']);

Inheritance
Inheritance is the concept of how an object is able to access fields and variables of their 'parent'. for example a worker inherits from a person object. A person for example has firstname and lastname, then a worker in addition has a job, but also still has a firstname and lastname. This is a basic concept in any programming language. In javascript we call inheritance prototyping. 

function Worker(firstname, lastname){
    this.firstname = firstname;
    this.lastname = lastname;
}

Worker.prototype.job = function(job) {
    this.job = job
    console.log(this.firstname + ' occupation is : '
        + this.job);
};

var john = new Worker('John', 'Doe');
john.job('Tradie');
console.log(john['job'])

Immediately Invoked Function Expressions
Another way to invoke a function or create a function anonymously is through IIFE, Immediately invoked function Expressions, which may have a funny looking syntax as per below. the last set of parenthesis is where the function is actually invoked.
(function(){
    console.log('IIFE')
})();
