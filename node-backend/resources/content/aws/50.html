Assumed:

  • You already have working knowledge of EC2 instances, Route 53, AWS Networking
  • If you have a landing page setup on your Lightsail, i.e if you hit visit your ip in the browser you have some sort of landing page

Lets dive in:

1. So as mentioned you should already have working knowledge of general AWS and should know what are EC2s etc, if that is the case setting up your LightSail is very similar, at the end of setting up your LightSail, you should have something similar to this: (the blacked out part should be your public ip)
2. Head over to Route 53 and click 'Create Hosted Zone'
3. Here are the 4 things you need to make sure are there:
NS - stands for Name Server - this is automatically generated by AWS
SOA - Start of Authority - generated by AWS too
A - Alias - you need to create this, first create a www.your domain.com and you should be able to pick IP address, and specify your IP, it would be the same as the one
above in your LightSail. Note: this needs to be public ip (blocked out in the image below)
A - Alias - second alis is just to point subdomain, so if people hits your_domain.com it should be the same as going to www. your_domain.com

4. Copy the NS values from above, these are your Name Servers
