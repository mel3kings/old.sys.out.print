• Download “ngrok”, put in your applications (https://ngrok.com/)
  • Create Symlink by:
    • cd /usr/local/bin
    • ln -s /Applications/ngrok ngrok
  • Run you local server’s mock server and publish it to a different port other than 8080
  • Run this command to publish your local server to ngrok:
    • ngrok http <port number>
